1. Install twint (pip3 install --user --upgrade git+https://github.com/twintproject/twint.git@origin/master#egg=twint)
2. Run the General markets notebook to get the tweets from popular tweeter financial analysers
3. Run the Scrap geo location notebook  with verified flag set to true once and set to false once to get tweets based on the geo location
4. Run the Verified on Tesla, BMW.. notebook  to get the tweets from verified users on Tesla, BMW and Volkswagen
5. Run the Verified Users on good ... notebook  to get the tweets from verified users on a range of good car brands
6. Run the Combine all files notebook  to combine all the csv files to get all the tweets used in the project

7. Run the Word count notebook to see the number of rows, word count and unique word count for our data
